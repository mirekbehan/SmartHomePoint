    Welcome to SmartHomePoint project pages.

    This pages are dedicated to Open Source SmartHomePoint project. We hope you like Smart Future and became contributor, therefore we propose the prototype applicaions and project description on this site. You could fork repository directly on GitHub or you could join to our team of Smart Home evangelists. We wonder to reveall the power and potencial of electronics & software openness to contribute better future in Intelligent Houses.

    Get started:

    $ git clone git@github.com:mirekbehan/SmartHomePoint.git
$ cd SmartHomePoint
$ mvn package
$ mvn -Djetty.port=9999 jetty:run
$ firefox http://localhost:9999
